Video Hosting Project
=====================
Tech: PHP + MySQL + HTML/CSS/JS
Theme: Red & Blue, Side Navigation, Grid layout, Search Bar
Permission System: roles (user/admin). Admin can manage users and videos.

How to run:
1. Place folder in your web server root (e.g., C:\xampp\htdocs\video_hosting or /var/www/html/video_hosting).
2. Create a MySQL database called 'video_hosting' and import 'database.sql'.
3. Ensure 'uploads' folder is writable by the webserver.
4. Open http://localhost/video_hosting/ in your browser.

Default admin creation:
- Use register page to create accounts, then change role manually in DB to 'admin' or run the SQL user insert in database.sql.
